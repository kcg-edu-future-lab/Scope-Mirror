## Clip Mirror
A tool to duplicate the clipped area of the desktop.

### Setup
- [Download the app](https://github.com/sakapon/Clip-Mirror/raw/master/Downloads/ClipMirror.Single-1.0.1.zip)

### Usage
- Execute ClipMirror.Single.exe

### System Requirements
- .NET Framework 4.5
